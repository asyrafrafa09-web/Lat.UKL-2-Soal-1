# Program untuk menghitung faktorial dari sebuah bilangan bulat positif.
Pengguna  memasukkan sebuah angka, lalu program menghitung hasil perkalian dari 1 sampai angka tersebut menggunakan perulangan for.
Nilai hasil disimpan dalam variabel bertipe integer.
Jika pengguna memasukkan bilangan negatif, program menampilkan pesan bahwa bilangan harus positif.
Program akan menampilkan hasil di layar.
